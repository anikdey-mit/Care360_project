Recommendations

Did you recommend this to your {{ session['grandparent_gender'] }}?

Fall due to wet floor.

The Elderly are more likely to fall due to the water present on the floor. This could cause injuries like fractures and muscle strains to your {{ session['grandparent_gender'] }}. Recommend {% if session['grandparent_gender'] == 'grandfather' %}him {% else %} her {% endif %} to mop the wet floors especially in the bathroom and kitchen as soon as she sees the floor wet due to water. Also, try to help {% if session['grandparent_gender'] == 'grandfather' %}him {% else %} her {% endif %} with mopping the wet floor.

Slip due to rugs and mats.

Slippery rugs, mats, or rolled up mats may cause slip and injury to the elderly. Recommend your {{ session['grandparent_gender'] }} to check and remove these products or make sure these are firmly secured and nonslip. There are non-slip products available from hardware stores that can be applied to floors as well.

Electrical Wires Around

The wires over the floors may cause falls. Recommend your {{ session['grandparent_gender'] }} to avoid having electrical extension cords running across floors by having an electrician install additional power points. If you are unable to install additional power points, then move the cord to run along the walls, not across the floor. The cords can be secured with self-adhesive cord clips that attach to the base of the wall.

Fall due to cracked path

Paths around your {{ session['grandparent_gender'] }}'s house are cracked. Tell your {{ session['grandparent_gender'] }} to get the cracked path around the house fixed as it may cause injury to {% if session['grandparent_gender'] == 'grandfather' %}him {% else %} her {% endif %}.

Unadequate light in the house

Ask your parents who can help to set some bulbs where the places have inadequate light. If LED lights can be used, then it can save a lot of energy as well.

No Grab Bar in Bathroom

Grab bars can help your {{ session['grandparent_gender'] }} to hold on to something while {% if session['grandparent_gender'] == 'grandfather' %}he {% else %} she {% endif %} is in the bathroom that reduces the risk of {% if session['grandparent_gender'] == 'grandfather' %}him {% else %} her {% endif %} fall in the bathroom. Recommend your {{ session['grandparent_gender'] }} to install safety grab bars within the bathroom that can help {% if session['grandparent_gender'] == 'grandfather' %}him {% else %}her {% endif %} to perform {% if session['grandparent_gender'] == 'grandfather' %}his {% else %}her {% endif %} day to day tasks while reducing the risk of falls.

Presence of Smoke Detector

Make sure you see smoke detectors in your grandparent's house in the living area, outside the kitchen. Also, see if it blinks red or not. If not let it know to your parents or grandparents.

Fire Extinguisher

Ask your {{ session['grandparent_gender'] }} to install a fire extinguisher in the kitchen and also tell {% if session['grandparent_gender'] == 'grandfather' %}him {% else %} her {% endif %}about its usage in stopping the fire in case of an emergency.
